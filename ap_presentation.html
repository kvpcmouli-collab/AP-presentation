What is Arithmetic Progression?

Definition

An Arithmetic Progression (AP) is a sequence of numbers where the difference between consecutive terms is constant.

Common Difference (d) = Constant
Example 1: 2, 5, 8, 11, 14, ...
Common difference d = 3
Example 2: 20, 15, 10, 5, 0, ...
Common difference d = -5

General Form of AP

If the first term is a and common difference is d:

a, a+d, a+2d, a+3d, a+4d, ...

Types of AP Based on Common Difference

  • d > 0: Increasing AP (3, 6, 9, 12, ...)
  • d < 0: Decreasing AP (10, 7, 4, 1, ...)
  • d = 0: Constant AP (5, 5, 5, 5, ...)

Important Properties

Property 1: Three Terms in AP

If a, b, c are in AP, then:

2b = a + c

Property 2: Arithmetic Mean

AM between two numbers a and b:

AM = (a + b)/2

Property 3: Selection of Terms

  • 3 terms: a-d, a, a+d
  • 5 terms: a-2d, a-d, a, a+d, a+2d
